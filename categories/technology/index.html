像程序员一样写作

8月 27 2020 探技 7 分钟 读完 (约 1015 字)

请原谅我的标题党，抑或你可以把它理解撑我拙劣的幽默感。

如果Hexo有一个方便的功能可以标记一篇技术文章的难易程度，我一定会给本篇加上“技术小白”的标签以免贻笑大方。尽管如此，本文还是不免会落入“新手不懂，高手无用”的尴尬境地，就只当作是单纯地为了记录下我开通这个博客的过程吧，仅此而已。

言归正传，首先要回答的问题是：究竟怎样才算是“像程序员一样”？我相信每个程序员都有自己心里面认为唯一正确的答案。在我看来，像程序员一样，未必就是全程看不到图形界面，只会一顿操作猛如虎地狂敲各种代码和命令。WordPress的官网上有一句保留至今的经典脚注：

CODE IS POETRY

翻译成中文我一样喜欢：

代码如诗

这句话完美地诠释了编程的一大特点（后面我会说到另一特点）——优雅地解决问题。

何为优雅？在我看来，无非就是举重若轻，游刃有余，用三五分的气力就能解决十分的问题，所谓事半功倍是也。当然优雅不是人人都能做到的，简单的两个字背后隐藏的其实是平时深厚的积累，甚至是很多人（包括我在内）不惜用一生去追求的目标。
